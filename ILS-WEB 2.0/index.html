ชีวิตในเอก
มีอะไร?

การใช้ชีวิตในเอกของเราก็ไม่ได้มีแค่เข้ามาเลคเชอร์อย่างเดียวนะ เมื่อเข้ามาแล้วจะมีรับน้องเพื่อกระชับสัมพันธ์กับเพื่อน ๆ และมีกิจกรรมของเอกนั่นคือห้องสมุดเคลื่อนที่ด้วย และก่อนที่ทุกคนจะขึ้นปี 4 ก็จะมีการฝึกงานในช่วงปิดเทอมหลังจบปี 3 ด้วย

ถามมา...ตอบไป

Q:ผมไม่ใช่สายกิจกรรม ไม่เข้ารับน้องได้ไหมครับ


A:การรับน้องมีขึ้นเพื่อกระชับความสัมพันธ์กับเพื่อนใหม่ที่เข้ามาด้วยกัน แต่ไม่บังคับนะ ไม่ต้องเข้าก็ได้ ไม่มีผลอะไรกับการใช้ชีวิตแน่นอน

Q:แล้วกิจกรรมรับน้องของคณะที่นี่มีโซตัสไหมครับ


A:ที่นี่ไม่มีโซตัส ไม่ต้องกังวลนะ

Q:ถ้าไม่เข้าร่วมกิจกรรมห้องสมุดเคลื่อนที่ของเอกจะมีปัญหาอะไรไหมครับ


A:จริงๆไม่ได้บังคับนะ แต่เข้าร่วมเพื่อเอาไว้ใส่พอร์ตและสั่งสมประสบการณ์ก็ดี

Q:ไม่ประสงค์ฝึกงานได้ไหมครับ


A:ไม่ได้ครับ เพราะต้องนำประสบการณ์ที่ได้จากการฝึกงานไปสัมมนาในวิชาบังคับ "การสัมมนาการฝึกปฏิบัติงาน" ซึ่งต้องนับหน่วยกิตก่อนจบด้วย
